# CHIP-8 Emulator

Your typical CHIP-8 copy. This one uses SDL2 as a rendering API.

## Dependencies

* `make`
* `SDL2`

## Usage

```shell
$ cd path/to/chip8
$ make
$ ./chip8 [../path/to/ROM]
$ make clean # optional
```

You can install `chip8` by running `sudo make install clean`.
The binary will be installed at `/usr/local/bin`
